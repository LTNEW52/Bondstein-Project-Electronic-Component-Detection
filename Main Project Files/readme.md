Before run the project, please read following carefully !!!

# REQUIREMENTS

This libaries are requried to run the project

1. ultralytics
2. streamlit
3. moviepy
4. streamlit-webrtc

do, pip install ultralytics streamlit moviepy streamlit-webrtc, also it is better to have python 3.10.11

# RUN THE PROJECT

To run the project, do, streamlit run app.py
