Welcome To Electronic Component Detector Project !!!

Before run the project, please read following carefully !!!

# REQUIREMENTS

This libaries are requried to run the project

1. ultralytics
2. streamlit
3. moviepy
4. streamlit-webrtc

do, pip install ultralytics streamlit moviepy streamlit-webrtc, also it is better to have python 3.10.11

# RUN THE PROJECT

To run the project, do, streamlit run app.py

# DIRECTORY GUIDE

1. DATASET RAW IMAGES Folder have images used to make dataset.
2. FINAL PROJECT DATASET AND MODEL TRAINING have final roboflow dataset and final model training colab notebook.
3. FINAL PROJECT TEST IMAGES AND VIDEOS have some test image and video outside the dataset.
4. PREVIOUS ATTEMPTS contains file related to previous YOLOv8 Model testing.
5. app.py runs streamlit
6. FINAL_PROJECT_BEST_MODEL.pt is the best.pt model from YOLOv11.
7. PERSONAL NOTE is a note containing some of my thoughts.
8. README.MD you are reading it!
